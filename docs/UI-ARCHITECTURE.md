# PSMS UI Architecture

## Primary Navigation

- Dashboard
- Participants
- Meetings
- Attendance
- Academics
- Alerts

## Participant 360 View

The Participant Profile is the center of PSMS.

Sections:

- Participant Information
- Program Participation
- Engagement
- Support
- Progress
- Development
- Intervention

## Dashboard Types

### Program Coordinator Dashboard
Operational participant management.

### Executive Dashboard
Program-level metrics and trends.

## Guiding Rule

Every screen must support one or more of the Five Pillars:

- Engagement
- Support
- Progress
- Development
- Intervention
