# Dashboard Specification

## Program Coordinator Dashboard

Focus: participant management.

Metrics:
- Active Participants
- Active Alerts
- Missing Attendance
- Missing Meetings
- Upcoming Follow-Ups
- Missing Semester Updates

## Executive Dashboard

Focus: program oversight.

Metrics:
- Total Participants
- Participants by Program
- Attendance Trends
- Meeting Trends
- Requirement Completion
- Active Interventions

The dashboard supports Engagement, Support, Progress, Development, and Intervention.
